#P-ONE Examples

This folder contains examples to walk through the basics of a P-ONE analysis. The following examples are available:

1- Generate Events with LeptonInjector, this outputs a h5 type file as currently public icetray doesn't contain an event generator
	1a - Convert the gernerator files to i3 format. In the future this will be one step
2- propagate leptons with PROPOSAL
3- propagate photons with clsim
