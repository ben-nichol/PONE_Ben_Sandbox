.. py:function:: generateGeometry(nCircles, DPS, strings, npmts, max_radius)

   Generate geometry for PONE Phase 1 detector configuration.

   :param nCircles: Number of concentric circles
   :type nCircles: int
   :param DPS: DOMs per string
   :type DPS: int
   :param strings: Number of strings per circle
   :type strings: list
   :param npmts: Number of PMTs per mDOM
   :type npmts: int
   :param max_radius: Maximum radius for outermost circle in meters
   :type max_radius: float

   :returns: Geometry map for all optical modules
   :rtype: I3OMGeoMap
