.. py:function:: makePartialGeometry(denseGeometry, domsUsed)

   Decomposes a geometry into a new geometry with only the DOMs specified.

   :param denseGeometry: An I3Geometry object representing the dense geometry.
   :type denseGeometry: I3Geometry
   :param domsUsed: A list of OMKey objects representing the DOMs to be
                    included in the new geometry.
   :type domsUsed: list

   :returns: A new I3Geometry object containing only the DOMs in domsUsed.
   :rtype: I3Geometry

   :raises ValueError: If an OMKey in domsUsed is not in denseGeometry.


.. py:function:: AddPOM(geomap, position, string, om, separation=0.0785, pom_radius=0.2159)

   Adds a POM to a I3OMGeoMap object at a specified location and IDs.

   POM consists of two spheres with correct PMT orientations.
   PMTs 1-8 are on face oriented in the +x direction.

   :param geomap: Object which will have the POM added to it.
   :type geomap: dataclasses.I3OMGeoMap
   :param position: [x,y,z] position of the centre of the POM.
   :type position: array-like
   :param string: String ID that the POM will be added to.
   :type string: int
   :param om: OM ID that the POM will be added to.
   :type om: int
   :param separation: Separation of the centre of the two
                      hemispheres in meters. Defaults to 0.0785.
   :type separation: float, optional
   :param pom_radius: Radius of each hemisphere in meters.
                      Defaults to 0.2159.
   :type pom_radius: float, optional


.. py:function:: AddPCAL(geomap, position, string, om, separation=0.0785, pcal_radius=0.2159)

.. py:function:: generateGeometry(xpositions, ypositions, depth, omsequence)

   Creates an I3OMGeoMap object from x,y coordinates and other parameters.

   :param xpositions: List of x positions for strings.
   :type xpositions: list
   :param ypositions: List of y positions for strings.
   :type ypositions: list
   :param depth: Array of depths for OMs.
   :type depth: array
   :param omsequence: List defining the OM sequence (POM/PCAL).
   :type omsequence: list

   :returns: Map of Optical Modules (OMs).
   :rtype: I3OMGeoMap

   :raises Exception: If unknown optical module type is specified.
