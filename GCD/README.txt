The GCD folder contains scripts that are used to build geometries for P-ONE.

Files:
Calib_and_DetStat_File.i3.gz = I3File with calibration information.

gcdHelper.py = contains some methods that read the Calibration files and builds things for the actual GCD. 

PONE_Phase1.py = Main python script that buils the standard 10 string geometry.
